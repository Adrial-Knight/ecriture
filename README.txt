Pour ajouter des articles ou des pages:
  - Allez dans le repertoire cible
  - Renommez le dernier éléments par le nombre d'éléments final
  - Utilisez une copie d'un éléments n'étant ni le premier ni le dernier
  - Renommez-le avec le bon numéro
  - Faites les raccords de liens sur les fichiers ajoutés
  - Faites le raccord sur l'éléments final
  - Ajoutez les articles manquants à l'Accueil
